# TMDB Movie Analysis

## Project Overview

This project explores the TMDB 5000 Movie Dataset using Python and Pandas.

The analysis focuses on:

- Revenue by Genre
- Number of Movies by Genre
- Average Rating by Genre

---

## Dataset

- TMDB 5000 Movie Dataset
- 4,803 movies

---

## Tools

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Key Findings

- Animation has the highest average revenue.
- Drama is the most common genre.
- High revenue does not necessarily mean high ratings.
